Dlight-pad_pd_1.8

Iphone/Ipod layout for TouchOSC software

Require the TouchOSC Editor to open/synchronise
Software available on editor's website http://hexler.net/software/touchosc


This version of the layout communicates with the Pure Data patch pd_dlight-pad_1.8.pd

TouchOSC Setup
	
	Network :
		Host				: IP address of the computer running D::Light
		Port (outgoing)		: 8888
		Port (incomming)	: 9999 

D::Light Setup

	OSC activated (green button)
	OSC Writer
		IP Address	: 127.0.0.1
		Port			: 7001
